The Tiwut Life Planner is a Python desktop application built using the Tkinter library. It provides a simple yet effective way to visualize your life goals, projects, and milestones as a dynamic, interactive Mind Map/Timeline.

Each "Node" represents a life point‚Äîa goal, project, or task‚Äîallowing you to track its progress, set deadlines, and add descriptions and external links.

‚ú® Key Features

  • Visual Goal Mapping: Organize your goals in a hierarchical tree structure, showing the relationship between major milestones and sub-tasks.
  • Progress Tracking: Use the built-in slider to track the completion percentage of any goal (Node), visually represented by a progress bar.
  • Detailed Node Editing: Easily edit the Title, Date/Deadline, Progress, URL/Link, and Description for any selected Node in the sidebar.
  • Link Integration: Store relevant links (e.g., project folders, documentation) directly in the node and open them with a single click.
  • Persistence: Save and load your entire life map using the custom .tiwut_timeline file format (JSON-based).
  • Intuitive Navigation: Click nodes to select them and use the canvas drag feature to move around your map.

üíª Getting Started

To run the application, you need Python installed on your system. Since the application uses the standard Python library tkinter, no external dependencies are needed.

1. Save the provided Python code as tiwut_planner.py.

2. Open your terminal or command prompt and run:

python tiwut_planner.py

The main window will open, starting with the root node "My Life".
